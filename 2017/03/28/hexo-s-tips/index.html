1、发布博客遇到报错

报错原因：根据日志定位到第5行，第1列的内容是：categories:Hexo。分类冒号后面需要加个空格再写内容！
解决办法：改成categories: Hexo即可！

报错原因：网络问题
解决办法：关闭VPN，重新发布即可。
